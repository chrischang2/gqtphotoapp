this is an android app for taking photos on a checklist, track progress, rename photos and folders, and automatically upload to dropbox

usage is primarily for quaity check photos for GQT shipments of recycled paper and metals

privacy policy:

no data is collected by this app, it just facilitates local photo capture and storage, and upload to personal dropbox app folder. 
it is up to the user to select which photos will be moved to the shared GQT folder
